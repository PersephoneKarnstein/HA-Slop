# Home Assistant Custom Integrations (Archived)

This repository has been split into individual repos for HACS compatibility. Each integration is now independently installable via HACS:

| Integration | Repository | HACS Install |
|---|---|---|
| **Bluesky Feed** | [ha-bluesky-feed](https://github.com/PersephoneKarnstein/ha-bluesky-feed) | Add as custom repo |
| **Citizen Incidents** | [ha-citizen-incidents](https://github.com/PersephoneKarnstein/ha-citizen-incidents) | Add as custom repo |
| **Seedtime Garden Planner** | [ha-seedtime](https://github.com/PersephoneKarnstein/ha-seedtime) | Add as custom repo |
| **Estrannaise HRT Monitor** | [ha-estrannaise](https://github.com/PersephoneKarnstein/ha-estrannaise) | Add as custom repo |

To install any integration via HACS: **HACS > Integrations > three-dot menu > Custom repositories** > paste the repo URL with category **Integration**.
